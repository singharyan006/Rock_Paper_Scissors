How to Play

Objective: Beat the computer in 5 rounds!

  • Rock beats Scissors
  • Paper beats Rock
  • Scissors beats Paper
  • Tie if both pick the same

Instructions: Click an icon (rock, paper, or scissors). The computer will randomly choose its move. The winner is announced each round. After 5 rounds, the final winner is declared.

About This Game

This Rock Paper Scissors game is a web-based implementation of the classic hand game between a player and the computer.

Developer: Aryan Singh

Tech Stack: HTML, CSS, JavaScript

Designed with a focus on aesthetics and user interaction. Future plans include adding sound effects, animations, and leaderboard functionality!
